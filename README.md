Landing bilíngue (PT/EN) com tema noir, explicando o pipeline FLUX.1-schnell que gera as 99 fotos de perfil do jogo.

## Rodar localmente

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # produção
```

## Stack

Next.js 14 (App Router) · React 18 · TypeScript · Tailwind CSS · i18n client-side (`components/i18n.tsx`) · Google Analytics 4.
